¿Qué son los keywords y como se usan?

Los keywords o palabras clave son palabras que describen el contenido de una página web. Son palabras que un usuario podría usar para buscar una página web o para describir el contenido de una página web.

Ejemplo:

laboratorio, análisis clínicos, investigación científica, diagnóstico médico, pruebas de laboratorio, química, microbiología, biología molecular

Deben estar separados por comas y no repetirse.

¿Qué son las meta description y para que se usan?

La meta description es un fragmento de texto que se muestra en los resultados de búsqueda de Google y que sirve para describir el contenido de una página web.

Ejemplo:

Laboratorio Clinico X: análisis clínicos, diagnóstico médico y pruebas de laboratorio confiables en química, microbiología y biología molecular. ¡Resultados precisos garantizados!

Deben estar tener una longitud máxima de 150 caracteres.

También es recomendable incluir al final frases como Descubre más u Obtén información.
